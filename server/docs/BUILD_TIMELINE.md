# Build Timeline

Compiled 2026-04-29.

Source: `git log --oneline --all --date=short` across `master`, `preview`, `dev`,
plus historical `claude/*` branches that have merged in.

Pure "Merge branch 'dev' into preview" promote-commits are listed at the end of
the run rather than inline (they ship code, but the underlying feature commit is
already in the table). Vercel/Railway deploy-trigger one-liners are folded into
a single row.
